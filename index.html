📖 Guide rapide Firebase (gratuit)

  1. Aller sur console.firebase.google.com
  2. Créer un projet → désactiver Google Analytics (optionnel)
  3. Dans le menu gauche : Realtime Database → Créer une base de données
  4. Choisir la région Europe (eu-west1) → Mode test → Activer
  5. ⚙ Paramètres du projet → onglet Général → Vos applications → icône </>
  6. Enregistrer l'application (nom au choix) → copier apiKey, databaseURL, projectId
  7. Coller ces 3 valeurs ci-dessus → cliquer Appliquer et connecter
  8. Partager l'URL de cette page avec vos agents — tout le monde partage les mêmes données !

⚠ Le mode test Firebase autorise la lecture/écriture pendant 30 jours. Ensuite, aller dans Realtime Database → Règles et remplacer par : { "rules": { ".read": true, ".write": true } }
